Photo Hosting — ImgBB

Photos are uploaded to ImgBB (free) and served as plain public URLs that display in any browser without authentication.
Get your free API key at api.imgbb.com — sign up, then click Get API key.

Steps to get your free API key:
1. Go to imgbb.com → Sign up (takes 1 minute)
2. Go to api.imgbb.com → Click Get API key
3. Copy the key and paste it above → click Save Key
4. Photo uploads will now work for everyone on all devices.

Google Sheets Sync

Task data syncs via Google Apps Script. Connection status:

Force Sync — Fix Different Tasks in Different Browsers

If different browsers are showing different tasks, run this on each browser. It compares your local data with the Sheet and pushes any newer local changes up. Then refresh the other browsers to pull the latest data down.

⚠ Important: Run on one browser at a time only. Wait for the confirmation message before running on another browser. Running on both simultaneously can cause conflicting writes.

🔄 Reset to Sheet Data

Clears all local data in this browser and pulls a fresh copy directly from Google Sheets. Use this on the other browser after you have run Force Sync on your master browser. This guarantees both browsers show identical data.

🔗 Update Apps Script URL

If you created a new deployment in Apps Script, paste the new /exec URL here and click Test & Save. The dashboard will use it immediately — no file editing or GitHub push needed.

How to get a new URL:
1. Open your Google Sheet → Extensions → Apps Script
2. Paste Code_v17.gs contents → Ctrl+S to save
3. Click Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone
4. Click Deploy → copy the new /exec URL → paste above
